# API Documentation

The Universal API provides programmatic access to uAsset data, minting, redemption, and more.

## Getting Started

The Universal API enables you to:

- Query uAsset information across all chains
- Access real-time price data
- Integrate minting and redemption flows
- Retrieve contract addresses and metadata
- Query proof of reserves data
