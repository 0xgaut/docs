# Universal Protocol Documentation

This is the official documentation for Universal Protocol, built with [Mintlify](https://mintlify.com).

## Overview

Universal is a wrapped asset protocol designed to enable trading for any token, on any chain. This documentation provides comprehensive guides for understanding and integrating with the Universal Protocol.

## Documentation Structure

### Introduction
- **Welcome** - Overview of Universal Protocol and what problems it solves
- **Protocol Overview** - How Universal works under the hood
- **Introduction to uAssets** - Understanding wrapped tokens
- **Proof of Reserves** - How backing is verified
- **FAQ** - Common questions

### Developers
- **Building with uAssets** - Integration guide
- **API Integration** - Using the Universal API
- **Issuance & Redemption API** - Minting and redemption
- **Contract Addresses** - Smart contract addresses across chains
- **Partnerships** - Partner with Universal
- **uAsset Logos** - Official brand assets

## Development

### Prerequisites
- Node.js 18+
- npm or yarn

### Install Mintlify CLI

```bash
npm i -g mintlify
```

### Run Locally

```bash
mintlify dev
```

The documentation will be available at `http://localhost:3000`.
