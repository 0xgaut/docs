## The Universal Mission

Universal makes all assets accessible across all supported chains, solving:

- **Limited Onchain Availability** - Many assets aren't available onchain
- **Liquidity Fragmentation** - Liquidity is split across multiple L1s and L2s
- **Poor User Experience** - Users struggle with bridges and fragmented execution

<Info>
  Universal enables non-EVM assets (like DOGE, XRP) to be wrapped and traded on any supported chain, expanding the DeFi ecosystem.
</Info>
