# Integrate the Universal API

The Universal API provides programmatic access to uAsset functionality.

## Overview

The Universal API enables developers to:

- Query uAsset data and metadata
- Access price feeds and market data
- Integrate seamlessly with the Universal Protocol
- Build custom applications on top of Universal
