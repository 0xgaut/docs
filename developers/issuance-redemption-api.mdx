# Integrate the Issuance & Redemption Universal API

The Issuance & Redemption API enables you to facilitate minting and redemption of uAssets within your application.

## Overview

This API allows you to:

- Enable users to mint uAssets from native assets
- Facilitate redemption of uAssets back to native assets
- Track the status of minting and redemption requests
- Integrate seamlessly with the merchant system

## Integration Flow

### Minting Flow

1. User initiates a mint request through your application
2. Your application calls the Issuance API
3. User deposits native assets to the provided address
4. Merchant verifies and mints uAssets
5. User receives uAssets on their chosen chain

### Redemption Flow

1. User initiates a redemption request
2. Your application calls the Redemption API
3. uAssets are burned
4. Custodian releases native assets
5. User receives native assets at specified address
