## Logo Usage

### In Your Application

- Display uAsset logos next to token symbols (e.g., uBTC, uSOL)
- Use appropriate sizes for different contexts (token lists, charts, etc.)
- Maintain adequate spacing around logos
- Ensure sufficient contrast with background

### File Formats

Official logos are typically provided in multiple formats:

- **SVG**: For scalable web use
- **PNG**: For general use with transparency
- **JPG**: For backgrounds and non-transparent contexts

## Brand Colors

When designing UI elements for uAssets, use Universal's brand colors:

- **Primary Purple**: `#7B3FF2`
- **Light Purple**: `#A78BFA`
- **Dark Purple**: `#5B21B6`
