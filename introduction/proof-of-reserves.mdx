# Proof of Reserves

Universal maintains transparent, verifiable proof that every uAsset is backed 1:1 by real assets held in custody.

## Overview

All assets are verifiably held by a qualified custodian, ensuring 1:1 backing for all wrapped uAssets.

## How It Works

Universal's Proof of Reserves system ensures that:

1. All assets are held by qualified custodians
2. Holdings can be verified in real-time
3. Circulating supply matches custodied amounts
4. Anyone can audit the reserves
